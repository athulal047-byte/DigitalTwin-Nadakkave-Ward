# Geographic Information Systems (GIS)

This directory manages the initial data acquisition and preprocessing phase of the pipeline.

## Structure
- `arcgis/`: Contains the Deep Learning evaluation models and height calculation methodologies.
- `qgis/`: Contains the initial OSM evaluation projects.
- `sample_data/`: Contains synthesized, open-source `.shp` files for pipeline testing.

## Data Privacy Warning
The official Nadakkave Ward Town Planner dataset used in the final visualization is **PROPRIETARY** to the Kozhikode Corporation. It must NOT be placed in this repository or committed to Git. All pipeline testing should be conducted using the files in `sample_data/`.
