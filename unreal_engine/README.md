# Unreal Engine 5 Environment

This directory contains the core 3D visualization project for the Digital Twin.

## Contents
- `Config/`: Standard UE5 `.ini` configuration files.
- `Content/`: All imported FBX assets, textures, semantic materials, and the UI Blueprint for the Web Browser Widget.
- `Source/`: (Empty in this Blueprint-only MVP, reserved for future C++ integrations).

## Setup
1. Ensure Unreal Engine 5.0+ is installed.
2. The **Web Browser Plugin** must be enabled in your engine plugins to render the React dashboard.
3. Open `DigitalTwin.uproject`.
