# Chapter Review Checklist

Before marking any chapter as complete, the Lead Author must verify:

- [ ] **Technical accuracy:** No invented features; true to workflow.
- [ ] **Grammar:** Proofread for syntax and academic tone.
- [ ] **Flow:** Logical progression of ideas.
- [ ] **References:** All citations match bibliography.
- [ ] **Figures:** Placeholders exist, captions describe the image.
- [ ] **Tables:** Placeholders exist, purpose is clear.
- [ ] **Consistency:** No contradiction with earlier chapters.
- [ ] **Formatting:** Follows Overleaf/LaTeX style guide.
