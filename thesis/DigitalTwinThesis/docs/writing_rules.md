# Writing Rules & Standards

## Style
- **Writing Style:** Formal academic English.
- **Tense:** Past tense for methodology (what was done), present tense for facts/literature.
- **Voice:** Passive voice preferred for technical descriptions, objective third-person.

## Terminology & Capitalization
- **Consistent Terms:** Unreal Engine 5 (not UE), Town Planner dataset, Web Browser Widget.
- **Capitalization:** Title Case for chapter headings, sentence case for figure captions.

## Citations
- **Citation Style:** Strict IEEE numeric standard (e.g., [1], [2]).

## Formatting
- **Figures:** Centered, caption below.
- **Tables:** Centered, caption above.
- **Equations:** Numbered right-aligned, variables defined immediately below.
